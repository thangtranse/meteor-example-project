# MeteorJS Example Project


#### 1. Starting

    meteor npm install
    meteor
    
- Running with port

        meteor --port <YOUR_PORT>

- Running with mongodb 

        export MONGO_URL=<MONGO_PATH_CONNECTIONN>
        mongo --port <YOUR_PORT>
